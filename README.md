### Create new problem set directory

```
cargo compete new <problem_number>
```

`<problem_number>`: Paste the tail of atcoder contest url
e.g. https://atcoder.jp/contests/abc300 -> abc300

### Run test cases locally
In contest directory, run following command.
For instance a.rs

```
cargo compete test a
```

### Submmit a code
```
cargo compete submit a
```
